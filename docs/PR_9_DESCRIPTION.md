# PR #9: Demo Video Script, Walkthrough, and Pitch Deck

## Summary
Adds hackathon demo materials for TrustyClaw to showcase real on-chain capabilities for the USDC Agent Hackathon submission.

## Files Added
- **docs/demo-video-script.md**: Professional 4-minute demo video script with scene-by-scene narration, on-screen text overlays, and background music suggestions
- **docs/demo-walkthrough.md**: Reproducible step-by-step guide with commands, expected outputs, and transaction hashes for live demos
- **docs/hackathon-deck.md**: 10-slide pitch deck covering Problem/Solution/Traction/Team for the USDC Agent Hackathon

## Demo Flow (4 minutes total)
1. **Opening (30s)**: Problem statement - "Trust in AI agents is broken"
2. **Demo Part 1 (1min)**: Renter finds agent via ML matching
3. **Demo Part 2 (1min)**: Autonomous mandate creation + USDC escrow
4. **Demo Part 3 (1min)**: Agent delivers, escrow releases, reputation updates
5. **Demo Part 4 (30s)**: On-chain verification with Solana Explorer links
6. **Closing (30s)**: Call to action + repository URL
